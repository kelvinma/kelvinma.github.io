## ğŸš€ Quick Start

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/kelvinma/kelvinma.github.io.git
   cd kelvinma.github.io
   ```

2. **Navigate to the Next.js project**
   ```bash
   cd portfolio-refresh
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run development server**
   ```bash
   npm run dev
   ```
   Navigate to http://localhost:3000

5. **Make changes**
   - Edit components in `src/components/` for UI updates
   - Modify app routes in `src/app/`
   - Update Tailwind styles in component files or `tailwind.config.*`
   - Add images and assets to `public/`

6. **Build for production**
   ```bash
   npm run build
   npm run start  # Test production build locally
   ```

### Deployment

This site is hosted on **GitHub Pages**. The Next.js application is built and the output is deployed to the root directory of the repository.

To deploy updates:
1. Build the Next.js app: `cd portfolio-refresh && npm run build`
2. Export static files to root directory (configured in `next.config.ts`)
3. Commit and push changes to the main branch
4. GitHub Pages automatically serves the updated site
